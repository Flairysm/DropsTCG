# DropsTCG
TCG arcade

## Getting Started

1. Install dependencies
   ```bash
   npm install
   ```

2. Start the app
   ```bash
   npx expo start
   ```

## Project Structure

This project uses [Expo Router](https://docs.expo.dev/router/introduction/) for file-based routing.
